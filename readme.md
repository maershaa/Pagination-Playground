# Pagination Playground

Учебный проект для изучения разных способов пагинации данных с backend API.  

Проект включает три типа пагинации:  
1. **Классическая пагинация** (page / limit)  
2. **Пагинация через кнопку "Load More"**  
3. **Infinite Scroll**  

---

## Технологии

- **Vite JS**  
- HTML / CSS / JavaScript (ES6+)  
- Fetch API для взаимодействия с backend
