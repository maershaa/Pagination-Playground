Image Search with Pagination

Приложение для поиска изображений — делает запросы к Pixabay API для получения изображений по ключевым словам. API поддерживает параметры page и per_page для пагинации.

Реализована классическая пагинация с выделением активной страницы . Для удобства пользователя одновременно отображается ограниченное окно ссылок на страницы — максимум 10. Если всего страниц больше 10, окно автоматически сдвигается так, чтобы текущая страница была примерно по центру. При достижении начала или конца диапазона отображаются корректные границы, чтобы первая и последняя страницы всегда были доступны.
