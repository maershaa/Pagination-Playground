Load more pagination

Пагинация через кнопку Load More:

Такой подход используется, когда данные загружаются порциями по страницам, а пользователь сам инициирует подгрузку следующей части контента.

При клике на кнопку Load More:
— увеличивается номер текущей страницы (currentPage);
— отправляется запрос на сервер с параметром страницы;
— полученные данные добавляются в конец уже существующего списка, без перезагрузки страницы.

Когда достигнута последняя страница, кнопку обычно скрывают или отключают, чтобы предотвратить лишние запросы к серверу.
